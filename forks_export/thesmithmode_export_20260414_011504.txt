# Task Service

Сервис для управления задачами с HTTP API на Go.

## Требования

- Go `1.23+`
- Docker и Docker Compose

## Быстрый запуск через Docker Compose

```bash
docker compose up --build
```

После запуска сервис будет доступен по адресу `http://localhost:8080`.

Если `postgres` уже запускался ранее со старой схемой, пересоздай volume:

```bash
docker compose down -v
docker compose up --build
```

Причина в том, что SQL-файлы из `migrations/` монтируются в `docker-entrypoint-initdb.d` и применяются только при инициализации пустого data volume.

## Swagger

Swagger UI:

```text
http://localhost:8080/swagger/
```

OpenAPI JSON:

```text
http://localhost:8080/swagger/openapi.json
```

## API

Базовый префикс API:

```text
/api/v1
```

### Задачи (Tasks)

- `POST /api/v1/tasks` — создать задачу
- `GET /api/v1/tasks` — список задач
- `GET /api/v1/tasks/{id}` — получить задачу по ID
- `PUT /api/v1/tasks/{id}` — обновить задачу
- `DELETE /api/v1/tasks/{id}` — удалить задачу

### Правила периодичности (Recurrences)

- `POST /api/v1/recurrences` — создать правило периодичности
- `GET /api/v1/recurrences` — список правил
- `GET /api/v1/recurrences/{id}` — получить правило по ID
- `PUT /api/v1/recurrences/{id}` — обновить правило
- `DELETE /api/v1/recurrences/{id}` — удалить правило
- `POST /api/v1/recurrences/{id}/generate` — сгенерировать задачи на диапазон дат

## Фича: периодичность задач

### Описание

Добавлена возможность задавать правила периодичности для автоматической генерации задач. Врачи могут определить расписание, по которому задачи будут создаваться автоматически.

### Типы периодичности

| Тип | Параметр | Описание |
|-----|----------|----------|
| `daily` | `every_n_days` (>= 1) | Каждый N-й день от даты начала |
| `monthly` | `day_of_month` (1-30) | Определённое число каждого месяца |
| `specific_dates` | `specific_dates` (массив дат) | Только на конкретные указанные даты |
| `even_days` | — | Чётные дни месяца (2, 4, 6, ..., 30) |
| `odd_days` | — | Нечётные дни месяца (1, 3, 5, ..., 29) |

### Примеры использования

**Создание ежедневного правила (каждые 2 дня):**
```bash
curl -X POST http://localhost:8080/api/v1/recurrences \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Обход пациентов",
    "description": "Утренний обход палаты 3",
    "recurrence_type": "daily",
    "every_n_days": 2,
    "start_date": "2026-01-01",
    "end_date": "2026-12-31"
  }'
```

**Создание ежемесячного правила (15-е число):**
```bash
curl -X POST http://localhost:8080/api/v1/recurrences \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Ежемесячный отчёт",
    "description": "Формирование отчётности",
    "recurrence_type": "monthly",
    "day_of_month": 15,
    "start_date": "2026-01-01",
    "end_date": "2026-12-31"
  }'
```

**Генерация задач на январь:**
```bash
curl -X POST http://localhost:8080/api/v1/recurrences/1/generate \
  -H "Content-Type: application/json" \
  -d '{
    "date_from": "2026-01-01",
    "date_to": "2026-01-31"
  }'
```

### Архитектура решения

- **Правило периодичности** (`task_recurrences`) — отдельная сущность, шаблон для генерации задач. Содержит title/description (копируются в задачи), тип и параметры расписания, временные границы действия.
- **Генерация задач** происходит двумя способами:
  1. **API-эндпоинт** `POST /recurrences/{id}/generate` — ручная генерация на произвольный диапазон дат.
  2. **Фоновый планировщик** — горутина внутри приложения, которая при запуске и затем каждый день в полночь UTC генерирует задачи на текущую дату по всем активным правилам.
- **Идемпотентность**: уникальный частичный индекс `(recurrence_id, scheduled_date)` в БД гарантирует, что повторный вызов генерации не создаёт дубликатов.
- **Обратная совместимость**: существующий API задач не изменён. Новые поля `scheduled_date` и `recurrence_id` — nullable и omitempty в JSON.

### Допущения и проектные решения

1. **Периодичность — отдельная сущность**, а не поле в задаче. Это позволяет менять правило без влияния на уже созданные задачи и чётко разделяет «шаблон» и «экземпляр».
2. **Сгенерированные задачи всегда создаются со статусом `new`**. Статус — свойство жизненного цикла конкретной задачи, а не шаблона.
3. **end_date не обязателен** (nullable). Если не задан — правило бессрочное, генерация ограничивается параметром `date_to` запроса.
4. **Максимальный диапазон генерации — 366 дней**. Защита от случайного создания огромного количества задач.
5. **day_of_month ограничен 1-30** (по ТЗ). Если в месяце меньше дней (февраль с 30-м) — этот месяц пропускается.
6. **Удаление правила не удаляет задачи** (ON DELETE SET NULL). Сгенерированные задачи — реальные рабочие единицы, которые могут быть уже в работе.
7. **Чётность/нечётность определяется по числу месяца**, а не по порядковому номеру дня в году.
8. **Ежедневные задачи выравниваются по start_date**: если `every_n_days=3` и `start_date=1 янв`, то задачи создаются 1, 4, 7, 10... января, даже если генерация запрошена с 5 числа.

package main

import (
	"context"
	"fmt"
	"log/slog"
	"net/http"
	"os"
	"os/signal"
	"syscall"
	"time"

	infrastructurepostgres "example.com/taskservice/internal/infrastructure/postgres"
	postgresrepo "example.com/taskservice/internal/repository/postgres"
	"example.com/taskservice/internal/scheduler"
	transporthttp "example.com/taskservice/internal/transport/http"
	swaggerdocs "example.com/taskservice/internal/transport/http/docs"
	httphandlers "example.com/taskservice/internal/transport/http/handlers"
	"example.com/taskservice/internal/usecase/recurrence"
	"example.com/taskservice/internal/usecase/task"
)

func main() {
	logger := slog.New(slog.NewTextHandler(os.Stdout, &slog.HandlerOptions{
		Level: slog.LevelInfo,
	}))

	cfg := loadConfig()

	ctx, stop := signal.NotifyContext(context.Background(), syscall.SIGINT, syscall.SIGTERM)
	defer stop()

	pool, err := infrastructurepostgres.Open(ctx, cfg.DatabaseDSN)
	if err != nil {
		logger.Error("open postgres", "error", err)
		os.Exit(1)
	}
	defer pool.Close()

	// Task layer.
	taskRepo := postgresrepo.New(pool)
	taskUsecase := task.NewService(taskRepo)
	taskHandler := httphandlers.NewTaskHandler(taskUsecase)

	// Recurrence layer.
	recurrenceRepo := postgresrepo.NewRecurrenceRepository(pool)
	recurrenceUsecase := recurrence.NewService(recurrenceRepo, taskRepo)
	recurrenceHandler := httphandlers.NewRecurrenceHandler(recurrenceUsecase)

	// Docs & router.
	docsHandler := swaggerdocs.NewHandler()
	router := transporthttp.NewRouter(taskHandler, recurrenceHandler, docsHandler)

	// Background scheduler: ежедневная генерация задач из правил периодичности.
	sched := scheduler.New(recurrenceUsecase, logger)
	go sched.Run(ctx)

	server := &http.Server{
		Addr:              cfg.HTTPAddr,
		Handler:           router,
		ReadHeaderTimeout: 5 * time.Second,
	}

	go func() {
		<-ctx.Done()

		shutdownCtx, cancel := context.WithTimeout(context.Background(), 10*time.Second)
		defer cancel()

		if err := server.Shutdown(shutdownCtx); err != nil {
			logger.Error("shutdown http server", "error", err)
		}
	}()

	logger.Info("http server started", "addr", cfg.HTTPAddr)

	if err := server.ListenAndServe(); err != nil && err != http.ErrServerClosed {
		logger.Error("listen and serve", "error", err)
		os.Exit(1)
	}
}

type config struct {
	HTTPAddr    string
	DatabaseDSN string
}

func loadConfig() config {
	cfg := config{
		HTTPAddr:    envOrDefault("HTTP_ADDR", ":8080"),
		DatabaseDSN: envOrDefault("DATABASE_DSN", "postgres://postgres:postgres@localhost:5432/taskservice?sslmode=disable"),
	}

	if cfg.DatabaseDSN == "" {
		panic(fmt.Errorf("DATABASE_DSN is required"))
	}

	return cfg
}

func envOrDefault(key, fallback string) string {
	if value := os.Getenv(key); value != "" {
		return value
	}

	return fallback
}

package recurrence

import "time"

// RecurrenceType определяет тип периодичности задачи.
type RecurrenceType string

const (
	TypeDaily         RecurrenceType = "daily"
	TypeMonthly       RecurrenceType = "monthly"
	TypeSpecificDates RecurrenceType = "specific_dates"
	TypeEvenDays      RecurrenceType = "even_days"
	TypeOddDays       RecurrenceType = "odd_days"
)

// Valid проверяет, является ли тип периодичности допустимым значением.
func (t RecurrenceType) Valid() bool {
	switch t {
	case TypeDaily, TypeMonthly, TypeSpecificDates, TypeEvenDays, TypeOddDays:
		return true
	default:
		return false
	}
}

// Recurrence описывает правило периодичности для генерации задач.
type Recurrence struct {
	ID             int64          `json:"id"`
	Title          string         `json:"title"`
	Description    string         `json:"description"`
	RecurrenceType RecurrenceType `json:"recurrence_type"`
	EveryNDays     int            `json:"every_n_days"`
	DayOfMonth     *int           `json:"day_of_month,omitempty"`
	SpecificDates  []time.Time    `json:"specific_dates,omitempty"`
	StartDate      time.Time      `json:"start_date"`
	EndDate        *time.Time     `json:"end_date,omitempty"`
	CreatedAt      time.Time      `json:"created_at"`
	UpdatedAt      time.Time      `json:"updated_at"`
}

// GenerateDates возвращает список дат, на которые должны быть созданы задачи,
// в пределах указанного диапазона [from, to].
// Диапазон обрезается по start_date и end_date правила.
func (r *Recurrence) GenerateDates(from, to time.Time) []time.Time {
	from = truncateToDate(from)
	to = truncateToDate(to)

	// Обрезаем диапазон по границам правила.
	effectiveFrom := maxDate(from, truncateToDate(r.StartDate))
	effectiveTo := to
	if r.EndDate != nil {
		effectiveTo = minDate(to, truncateToDate(*r.EndDate))
	}

	if effectiveFrom.After(effectiveTo) {
		return nil
	}

	switch r.RecurrenceType {
	case TypeDaily:
		return generateDaily(truncateToDate(r.StartDate), r.EveryNDays, effectiveFrom, effectiveTo)
	case TypeMonthly:
		dom := 1
		if r.DayOfMonth != nil {
			dom = *r.DayOfMonth
		}
		return generateMonthly(dom, effectiveFrom, effectiveTo)
	case TypeSpecificDates:
		return filterDatesInRange(r.SpecificDates, effectiveFrom, effectiveTo)
	case TypeEvenDays:
		return generateByDayParity(effectiveFrom, effectiveTo, true)
	case TypeOddDays:
		return generateByDayParity(effectiveFrom, effectiveTo, false)
	default:
		return nil
	}
}

// generateDaily генерирует даты с интервалом everyN дней, начиная от startDate.
// Возвращает только даты в диапазоне [from, to].
func generateDaily(startDate time.Time, everyN int, from, to time.Time) []time.Time {
	if everyN < 1 {
		everyN = 1
	}

	// Находим первую дату >= from, выровненную по интервалу от startDate.
	daysSinceStart := int(from.Sub(startDate).Hours() / 24)
	if daysSinceStart < 0 {
		daysSinceStart = 0
	}

	remainder := daysSinceStart % everyN
	var first time.Time
	if remainder == 0 {
		first = startDate.AddDate(0, 0, daysSinceStart)
	} else {
		first = startDate.AddDate(0, 0, daysSinceStart+(everyN-remainder))
	}

	var dates []time.Time
	for d := first; !d.After(to); d = d.AddDate(0, 0, everyN) {
		dates = append(dates, d)
	}

	return dates
}

// generateMonthly генерирует даты с указанным числом месяца в диапазоне [from, to].
// Пропускает месяцы, в которых нет такого числа (напр. 30 февраля).
func generateMonthly(dayOfMonth int, from, to time.Time) []time.Time {
	var dates []time.Time

	year, month, _ := from.Date()
	for {
		candidate := time.Date(year, month, dayOfMonth, 0, 0, 0, 0, time.UTC)

		// Проверяем, что Go не перенёс дату в следующий месяц
		// (time.Date(2026, 2, 30, ...) → 2 марта, а нам нужно пропустить).
		if candidate.Month() == month && candidate.Day() == dayOfMonth {
			if !candidate.Before(from) && !candidate.After(to) {
				dates = append(dates, candidate)
			}
		}

		// Переходим к следующему месяцу.
		month++
		if month > 12 {
			month = 1
			year++
		}

		// Проверяем, не вышли ли за пределы диапазона.
		if time.Date(year, month, 1, 0, 0, 0, 0, time.UTC).After(to) {
			break
		}
	}

	return dates
}

// filterDatesInRange возвращает только те даты из списка, которые попадают в [from, to].
func filterDatesInRange(dates []time.Time, from, to time.Time) []time.Time {
	var result []time.Time
	for _, d := range dates {
		d = truncateToDate(d)
		if !d.Before(from) && !d.After(to) {
			result = append(result, d)
		}
	}
	return result
}

// generateByDayParity генерирует даты чётных или нечётных дней месяца в диапазоне [from, to].
// even=true: чётные дни (2, 4, 6, ..., 30)
// even=false: нечётные дни (1, 3, 5, ..., 29)
func generateByDayParity(from, to time.Time, even bool) []time.Time {
	var dates []time.Time
	for d := from; !d.After(to); d = d.AddDate(0, 0, 1) {
		day := d.Day()
		isEven := day%2 == 0
		if isEven == even {
			dates = append(dates, d)
		}
	}
	return dates
}

func truncateToDate(t time.Time) time.Time {
	y, m, d := t.Date()
	return time.Date(y, m, d, 0, 0, 0, 0, time.UTC)
}

func maxDate(a, b time.Time) time.Time {
	if a.After(b) {
		return a
	}
	return b
}

func minDate(a, b time.Time) time.Time {
	if a.Before(b) {
		return a
	}
	return b
}

package postgres

import (
	"context"
	"fmt"

	"github.com/jackc/pgx/v5/pgxpool"
)

func Open(ctx context.Context, dsn string) (*pgxpool.Pool, error) {
	if dsn == "" {
		return nil, fmt.Errorf("database dsn is empty")
	}

	cfg, err := pgxpool.ParseConfig(dsn)
	if err != nil {
		return nil, err
	}

	pool, err := pgxpool.NewWithConfig(ctx, cfg)
	if err != nil {
		return nil, err
	}

	if err := pool.Ping(ctx); err != nil {
		pool.Close()
		return nil, err
	}

	return pool, nil
}

// CreateBatch пакетно создаёт задачи с идемпотентностью.
// Использует ON CONFLICT DO NOTHING для пропуска дубликатов
// по уникальному индексу (recurrence_id, scheduled_date).
// Возвращает только реально созданные задачи.
func (r *Repository) CreateBatch(ctx context.Context, tasks []*taskdomain.Task) ([]taskdomain.Task, error) {
	if len(tasks) == 0 {
		return nil, nil
	}

	// Строим batch INSERT с VALUES и ON CONFLICT DO NOTHING RETURNING.
	var sb strings.Builder
	sb.WriteString(`
		WITH inserted AS (
			INSERT INTO tasks (title, description, status, scheduled_date, recurrence_id, created_at, updated_at)
			VALUES `)

	args := make([]any, 0, len(tasks)*7)
	for i, task := range tasks {
		if i > 0 {
			sb.WriteString(", ")
		}
		base := i*7 + 1
		sb.WriteString(fmt.Sprintf("($%d, $%d, $%d, $%d, $%d, $%d, $%d)",
			base, base+1, base+2, base+3, base+4, base+5, base+6))
		args = append(args,
			task.Title, task.Description, task.Status,
			task.ScheduledDate, task.RecurrenceID,
			task.CreatedAt, task.UpdatedAt,
		)
	}

	sb.WriteString(`
			ON CONFLICT (recurrence_id, scheduled_date) WHERE recurrence_id IS NOT NULL DO NOTHING
			RETURNING id, title, description, status, scheduled_date, recurrence_id, created_at, updated_at
		)
		SELECT * FROM inserted
		ORDER BY scheduled_date`)

	rows, err := r.pool.Query(ctx, sb.String(), args...)
	if err != nil {
		return nil, err
	}
	defer rows.Close()

	created := make([]taskdomain.Task, 0, len(tasks))
	for rows.Next() {
		task, err := scanTask(rows)
		if err != nil {
			return nil, err
		}
		created = append(created, *task)
	}

	if err := rows.Err(); err != nil {
		return nil, err
	}

	return created, nil
}

package scheduler

import (
	"context"
	"log/slog"
	"time"

	recusecase "example.com/taskservice/internal/usecase/recurrence"
)

// Scheduler — фоновый планировщик, который ежедневно генерирует
// задачи из активных правил периодичности на текущую дату.
type Scheduler struct {
	usecase recusecase.Usecase
	logger  *slog.Logger
	now     func() time.Time
}

// New создаёт новый экземпляр планировщика.
func New(usecase recusecase.Usecase, logger *slog.Logger) *Scheduler {
	return &Scheduler{
		usecase: usecase,
		logger:  logger,
		now:     func() time.Time { return time.Now().UTC() },
	}
}

// Run запускает планировщик. Блокирует до отмены контекста.
// Генерация происходит сразу при запуске и затем каждый день в полночь UTC.
func (s *Scheduler) Run(ctx context.Context) {
	s.logger.Info("планировщик запущен")

	// Генерируем задачи на сегодня при старте.
	s.generateForToday(ctx)

	for {
		nextMidnight := s.nextMidnightUTC()
		timer := time.NewTimer(time.Until(nextMidnight))

		select {
		case <-ctx.Done():
			timer.Stop()
			s.logger.Info("планировщик остановлен")
			return
		case <-timer.C:
			s.generateForToday(ctx)
		}
	}
}

func (s *Scheduler) generateForToday(ctx context.Context) {
	today := s.truncateToDate(s.now())

	s.logger.Info("генерация задач на дату", "date", today.Format("2006-01-02"))

	tasks, err := s.usecase.GenerateAll(ctx, today, today)
	if err != nil {
		s.logger.Error("ошибка генерации задач", "error", err)
		return
	}

	s.logger.Info("задачи сгенерированы", "date", today.Format("2006-01-02"), "count", len(tasks))
}

func (s *Scheduler) nextMidnightUTC() time.Time {
	now := s.now()
	tomorrow := now.AddDate(0, 0, 1)
	return time.Date(tomorrow.Year(), tomorrow.Month(), tomorrow.Day(), 0, 0, 0, 0, time.UTC)
}

func (s *Scheduler) truncateToDate(t time.Time) time.Time {
	y, m, d := t.Date()
	return time.Date(y, m, d, 0, 0, 0, 0, time.UTC)
}

func (dto *recurrenceMutationDTO) toMutationInput() (recusecase.MutationInput, error) {

		Title:          dto.Title,
		Description:    dto.Description,
		RecurrenceType: recdomain.RecurrenceType(dto.RecurrenceType),
		EveryNDays:     dto.EveryNDays,
		DayOfMonth:     dto.DayOfMonth,
		StartDate:      startDate,
	}

		endDate, err := time.Parse(dateLayout, *dto.EndDate)
		if err != nil {
			return recusecase.MutationInput{}, err
		}
		input.EndDate = &endDate
	}

		dates := make([]time.Time, 0, len(dto.SpecificDates))
		for _, s := range dto.SpecificDates {
			d, err := time.Parse(dateLayout, s)
			if err != nil {
				return recusecase.MutationInput{}, err
			}
			dates = append(dates, d)
		}
		input.SpecificDates = dates
	}

	return input, nil
}

type RecurrenceHandler struct {
	usecase recusecase.Usecase
}

// NewRecurrenceHandler создаёт новый обработчик правил периодичности.
func NewRecurrenceHandler(usecase recusecase.Usecase) *RecurrenceHandler {
	return &RecurrenceHandler{usecase: usecase}
}

func (h *RecurrenceHandler) Create(w http.ResponseWriter, r *http.Request) {
	var req recurrenceMutationDTO
	if err := decodeJSON(r, &req); err != nil {
		writeError(w, http.StatusBadRequest, err)
		return
	}

	input, err := req.toMutationInput()
	if err != nil {
		writeError(w, http.StatusBadRequest, err)
		return
	}

	created, err := h.usecase.Create(r.Context(), input)
	if err != nil {
		writeRecurrenceUsecaseError(w, err)
		return
	}

	writeJSON(w, http.StatusCreated, newRecurrenceDTO(created))
}

func (h *RecurrenceHandler) GetByID(w http.ResponseWriter, r *http.Request) {

	if err != nil {
		writeError(w, http.StatusBadRequest, err)
		return
	}

	rec, err := h.usecase.GetByID(r.Context(), id)
	if err != nil {
		writeRecurrenceUsecaseError(w, err)
		return
	}

	writeJSON(w, http.StatusOK, newRecurrenceDTO(rec))
}

	id, err := getIDFromRequest(r)

		writeError(w, http.StatusBadRequest, err)
		return
	}

	updated, err := h.usecase.Update(r.Context(), id, input)
	if err != nil {
		writeRecurrenceUsecaseError(w, err)
		return
	}

	writeJSON(w, http.StatusOK, newRecurrenceDTO(updated))
}

func (h *RecurrenceHandler) Delete(w http.ResponseWriter, r *http.Request) {

		return

	if err != nil {
		writeRecurrenceUsecaseError(w, err)
		return
	}

	response := make([]recurrenceDTO, 0, len(recs))
	for i := range recs {
		response = append(response, newRecurrenceDTO(&recs[i]))
	}

	writeJSON(w, http.StatusOK, response)
}

func (h *RecurrenceHandler) Generate(w http.ResponseWriter, r *http.Request) {
	id, err := getIDFromRequest(r)
	if err != nil {
		writeError(w, http.StatusBadRequest, err)
		return
	}

	var req generateRequestDTO
	if err := decodeJSON(r, &req); err != nil {
		writeError(w, http.StatusBadRequest, err)
		return
	}

	dateFrom, err := time.Parse(dateLayout, req.DateFrom)
	if err != nil {
		writeError(w, http.StatusBadRequest, errors.New("invalid date_from format, expected YYYY-MM-DD"))
		return
	}

	dateTo, err := time.Parse(dateLayout, req.DateTo)
	if err != nil {
		writeError(w, http.StatusBadRequest, errors.New("invalid date_to format, expected YYYY-MM-DD"))
		return
	}

	tasks, err := h.usecase.Generate(r.Context(), id, dateFrom, dateTo)
	if err != nil {
		writeRecurrenceUsecaseError(w, err)
		return
	}

	taskDTOs := make([]taskDTO, 0, len(tasks))
	for i := range tasks {
		taskDTOs = append(taskDTOs, newTaskDTO(&tasks[i]))
	}

	writeJSON(w, http.StatusCreated, generateResponseDTO{
		CreatedCount: len(taskDTOs),
		Tasks:        taskDTOs,
	})
}

func writeRecurrenceUsecaseError(w http.ResponseWriter, err error) {
	switch {
	case errors.Is(err, recdomain.ErrNotFound):
		writeError(w, http.StatusNotFound, err)
	case errors.Is(err, recusecase.ErrInvalidInput):
		writeError(w, http.StatusBadRequest, err)
	default:
		slog.Error("internal error in recurrence handler", "error", err)
		writeError(w, http.StatusInternalServerError, errors.New("internal server error"))
	}
}

import (

	taskdomain "example.com/taskservice/internal/domain/task"

type TaskHandler struct {
	usecase taskusecase.Usecase
}

func NewTaskHandler(usecase taskusecase.Usecase) *TaskHandler {
	return &TaskHandler{usecase: usecase}
}

func (h *TaskHandler) Create(w http.ResponseWriter, r *http.Request) {
	var req taskMutationDTO
	if err := decodeJSON(r, &req); err != nil {
		writeError(w, http.StatusBadRequest, err)
		return
	}

	created, err := h.usecase.Create(r.Context(), taskusecase.CreateInput{
		Title:       req.Title,
		Description: req.Description,
		Status:      req.Status,
	})
	if err != nil {
		writeUsecaseError(w, err)
		return
	}

	writeJSON(w, http.StatusCreated, newTaskDTO(created))
}

func (h *TaskHandler) GetByID(w http.ResponseWriter, r *http.Request) {
	id, err := getIDFromRequest(r)
	if err != nil {
		writeError(w, http.StatusBadRequest, err)
		return
	}

	task, err := h.usecase.GetByID(r.Context(), id)
	if err != nil {
		writeUsecaseError(w, err)
		return
	}

	writeJSON(w, http.StatusOK, newTaskDTO(task))
}

func (h *TaskHandler) Update(w http.ResponseWriter, r *http.Request) {
	id, err := getIDFromRequest(r)
	if err != nil {
		writeError(w, http.StatusBadRequest, err)
		return
	}

	var req taskMutationDTO
	if err := decodeJSON(r, &req); err != nil {
		writeError(w, http.StatusBadRequest, err)
		return
	}

	updated, err := h.usecase.Update(r.Context(), id, taskusecase.UpdateInput{
		Title:       req.Title,
		Description: req.Description,
		Status:      req.Status,
	})
	if err != nil {
		writeUsecaseError(w, err)
		return
	}

	writeJSON(w, http.StatusOK, newTaskDTO(updated))
}

func (h *TaskHandler) Delete(w http.ResponseWriter, r *http.Request) {
	id, err := getIDFromRequest(r)
	if err != nil {
		writeError(w, http.StatusBadRequest, err)
		return
	}

	if err := h.usecase.Delete(r.Context(), id); err != nil {
		writeUsecaseError(w, err)
		return
	}

	w.WriteHeader(http.StatusNoContent)
}

func (h *TaskHandler) List(w http.ResponseWriter, r *http.Request) {
	tasks, err := h.usecase.List(r.Context())
	if err != nil {
		writeUsecaseError(w, err)
		return
	}

	response := make([]taskDTO, 0, len(tasks))
	for i := range tasks {
		response = append(response, newTaskDTO(&tasks[i]))
	}

	writeJSON(w, http.StatusOK, response)
}

func getIDFromRequest(r *http.Request) (int64, error) {
	rawID := mux.Vars(r)["id"]
	if rawID == "" {
		return 0, errors.New("missing task id")
	}

	id, err := strconv.ParseInt(rawID, 10, 64)
	if err != nil {
		return 0, errors.New("invalid task id")
	}

	if id <= 0 {
		return 0, errors.New("invalid task id")
	}

	return id, nil
}

func decodeJSON(r *http.Request, dst any) error {
	decoder := json.NewDecoder(r.Body)
	decoder.DisallowUnknownFields()

	if err := decoder.Decode(dst); err != nil {
		return err
	}

	return nil
}

func writeUsecaseError(w http.ResponseWriter, err error) {
	switch {
	case errors.Is(err, taskdomain.ErrNotFound):
		writeError(w, http.StatusNotFound, err)
	case errors.Is(err, taskusecase.ErrInvalidInput):
		writeError(w, http.StatusBadRequest, err)
	default:
		writeError(w, http.StatusInternalServerError, err)
	}
}

func writeError(w http.ResponseWriter, status int, err error) {
	writeJSON(w, status, map[string]string{
		"error": err.Error(),
	})
}

func writeJSON(w http.ResponseWriter, status int, payload any) {
	w.Header().Set("Content-Type", "application/json")
	w.WriteHeader(status)

	_ = json.NewEncoder(w).Encode(payload)
}

// Generate создаёт задачи по одному правилу периодичности на указанный диапазон дат.
// Идемпотентна: повторный вызов не создаёт дубликатов благодаря уникальному индексу в БД.
func (s *Service) Generate(ctx context.Context, id int64, dateFrom, dateTo time.Time) ([]taskdomain.Task, error) {
	if id <= 0 {
		return nil, fmt.Errorf("%w: id must be positive", ErrInvalidInput)
	}

	if err := validateDateRange(dateFrom, dateTo); err != nil {
		return nil, err
	}

	rec, err := s.repo.GetByID(ctx, id)
	if err != nil {
		return nil, err
	}

	return s.generateTasksForRecurrence(ctx, rec, dateFrom, dateTo)
}

// GenerateAll создаёт задачи по всем активным правилам периодичности на указанный диапазон дат.
// Используется фоновым планировщиком для ежедневной генерации.
// Загружает только правила, активные на указанную дату (start_date <= dateTo и end_date >= dateFrom или end_date IS NULL).
// Все задачи создаются одним вызовом CreateBatch — атомарно на уровне одного INSERT-запроса.
func (s *Service) GenerateAll(ctx context.Context, dateFrom, dateTo time.Time) ([]taskdomain.Task, error) {
	if err := validateDateRange(dateFrom, dateTo); err != nil {
		return nil, err
	}

	recs, err := s.repo.ListActive(ctx, dateFrom)
	if err != nil {
		return nil, err
	}

	now := s.now()
	var allPending []*taskdomain.Task
	for i := range recs {
		pending := buildTasksForRecurrence(&recs[i], dateFrom, dateTo, now)
		allPending = append(allPending, pending...)
	}

	if len(allPending) == 0 {
		return nil, nil
	}

	return s.taskRepo.CreateBatch(ctx, allPending)
}

func (s *Service) generateTasksForRecurrence(ctx context.Context, rec *recdomain.Recurrence, dateFrom, dateTo time.Time) ([]taskdomain.Task, error) {
	tasks := buildTasksForRecurrence(rec, dateFrom, dateTo, s.now())
	if len(tasks) == 0 {
		return nil, nil
	}

	return s.taskRepo.CreateBatch(ctx, tasks)
}

// buildTasksForRecurrence — чистая функция без побочных эффектов.
// Строит список задач по правилу периодичности на указанный диапазон дат.
func buildTasksForRecurrence(rec *recdomain.Recurrence, dateFrom, dateTo time.Time, now time.Time) []*taskdomain.Task {
	dates := rec.GenerateDates(dateFrom, dateTo)
	if len(dates) == 0 {
		return nil
	}

	tasks := make([]*taskdomain.Task, 0, len(dates))
	for _, d := range dates {
		sd := d
		rid := rec.ID
		tasks = append(tasks, &taskdomain.Task{
			Title:         rec.Title,
			Description:   rec.Description,
			Status:        taskdomain.StatusNew,
			ScheduledDate: &sd,
			RecurrenceID:  &rid,
			CreatedAt:     now,
			UpdatedAt:     now,
		})
	}

	return tasks
}

func validateDateRange(dateFrom, dateTo time.Time) error {
	if dateTo.Before(dateFrom) {
		return fmt.Errorf("%w: date_from must not be after date_to", ErrInvalidInput)
	}

	days := int(dateTo.Sub(dateFrom).Hours()/24) + 1
	if days > maxGenerateRangeDays {
		return fmt.Errorf("%w: date range must not exceed %d days", ErrInvalidInput, maxGenerateRangeDays)
	}

	return nil
}

// validateMutationInput валидирует и нормализует входные данные для создания/обновления правила.
func validateMutationInput(input MutationInput) (MutationInput, error) {
	input.Title = strings.TrimSpace(input.Title)
	input.Description = strings.TrimSpace(input.Description)

	if input.Title == "" {
		return MutationInput{}, fmt.Errorf("%w: title is required", ErrInvalidInput)
	}

	if !input.RecurrenceType.Valid() {
		return MutationInput{}, fmt.Errorf("%w: invalid recurrence type", ErrInvalidInput)
	}

	if err := validateTypeSpecificParams(input.RecurrenceType, input.DayOfMonth, input.SpecificDates); err != nil {
		return MutationInput{}, err
	}

	if input.StartDate.IsZero() {
		return MutationInput{}, fmt.Errorf("%w: start_date is required", ErrInvalidInput)
	}

	if input.EndDate != nil && input.EndDate.Before(input.StartDate) {
		return MutationInput{}, fmt.Errorf("%w: end_date must not be before start_date", ErrInvalidInput)
	}

	if input.EveryNDays < 1 {
		input.EveryNDays = 1
	}

	return input, nil
}

func validateTypeSpecificParams(recType recdomain.RecurrenceType, dayOfMonth *int, specificDates []time.Time) error {
	switch recType {
	case recdomain.TypeDaily:
		// every_n_days нормализуется в validateMutationInput (0 → 1).
	case recdomain.TypeMonthly:
		if dayOfMonth == nil {
			return fmt.Errorf("%w: day_of_month is required for monthly recurrence", ErrInvalidInput)
		}
		if *dayOfMonth < 1 || *dayOfMonth > 30 {
			return fmt.Errorf("%w: day_of_month must be between 1 and 30", ErrInvalidInput)
		}
	case recdomain.TypeSpecificDates:
		if len(specificDates) == 0 {
			return fmt.Errorf("%w: specific_dates must not be empty", ErrInvalidInput)
		}
		if len(specificDates) > maxSpecificDates {
			return fmt.Errorf("%w: specific_dates must not exceed %d entries", ErrInvalidInput, maxSpecificDates)
		}
	case recdomain.TypeEvenDays, recdomain.TypeOddDays:
		// Не требуют дополнительных параметров.
	}
	return nil
}

package task

import (
	"context"
	"fmt"
	"strings"
	"time"

	taskdomain "example.com/taskservice/internal/domain/task"
)

type Service struct {
	repo Repository
	now  func() time.Time
}

func NewService(repo Repository) *Service {
	return &Service{
		repo: repo,
		now:  func() time.Time { return time.Now().UTC() },
	}
}

func (s *Service) Create(ctx context.Context, input CreateInput) (*taskdomain.Task, error) {
	normalized, err := validateCreateInput(input)
	if err != nil {
		return nil, err
	}

	model := &taskdomain.Task{
		Title:       normalized.Title,
		Description: normalized.Description,
		Status:      normalized.Status,
	}
	now := s.now()
	model.CreatedAt = now
	model.UpdatedAt = now

	created, err := s.repo.Create(ctx, model)
	if err != nil {
		return nil, err
	}

	return created, nil
}

func (s *Service) GetByID(ctx context.Context, id int64) (*taskdomain.Task, error) {
	if id <= 0 {
		return nil, fmt.Errorf("%w: id must be positive", ErrInvalidInput)
	}

	return s.repo.GetByID(ctx, id)
}

func (s *Service) Update(ctx context.Context, id int64, input UpdateInput) (*taskdomain.Task, error) {
	if id <= 0 {
		return nil, fmt.Errorf("%w: id must be positive", ErrInvalidInput)
	}

	normalized, err := validateUpdateInput(input)
	if err != nil {
		return nil, err
	}

	model := &taskdomain.Task{
		ID:          id,
		Title:       normalized.Title,
		Description: normalized.Description,
		Status:      normalized.Status,
		UpdatedAt:   s.now(),
	}

	updated, err := s.repo.Update(ctx, model)
	if err != nil {
		return nil, err
	}

	return updated, nil
}

func (s *Service) Delete(ctx context.Context, id int64) error {
	if id <= 0 {
		return fmt.Errorf("%w: id must be positive", ErrInvalidInput)
	}

	return s.repo.Delete(ctx, id)
}

func (s *Service) List(ctx context.Context) ([]taskdomain.Task, error) {
	return s.repo.List(ctx)
}

func validateCreateInput(input CreateInput) (CreateInput, error) {
	input.Title = strings.TrimSpace(input.Title)
	input.Description = strings.TrimSpace(input.Description)

	if input.Title == "" {
		return CreateInput{}, fmt.Errorf("%w: title is required", ErrInvalidInput)
	}

	if input.Status == "" {
		input.Status = taskdomain.StatusNew
	}

	if !input.Status.Valid() {
		return CreateInput{}, fmt.Errorf("%w: invalid status", ErrInvalidInput)
	}

	return input, nil
}
