# Task Service API

Сервис для управления задачами с HTTP API на Go. Поддерживает периодические задачи с автоматической генерацией.

## Требования

- Go `1.23+`
- Docker и Docker Compose

## Быстрый запуск

```bash
docker compose up --build
```

Сервис будет доступен по адресу `http://localhost:8080`.

Если `postgres` уже запускался ранее со старой схемой, пересоздай volume:

```bash
docker compose down -v
docker compose up --build
```

## Swagger

- Swagger UI: [http://localhost:8080/swagger/](http://localhost:8080/swagger/)
- OpenAPI JSON: [http://localhost:8080/swagger/openapi.json](http://localhost:8080/swagger/openapi.json)

## API

Базовый префикс: `/api/v1`

### Задачи

| Метод | URL | Описание |
|-------|-----|----------|
| POST | `/api/v1/tasks` | Создать задачу |
| GET | `/api/v1/tasks` | Список задач |
| GET | `/api/v1/tasks/{id}` | Получить задачу по ID |
| PUT | `/api/v1/tasks/{id}` | Обновить задачу |
| DELETE | `/api/v1/tasks/{id}` | Удалить задачу |

### Правила периодичности

| Метод | URL | Описание |
|-------|-----|----------|
| POST | `/api/v1/recurrence-rules` | Создать правило |
| GET | `/api/v1/recurrence-rules` | Список правил |
| GET | `/api/v1/recurrence-rules/{id}` | Получить правило по ID |
| PUT | `/api/v1/recurrence-rules/{id}` | Обновить правило |
| DELETE | `/api/v1/recurrence-rules/{id}` | Удалить правило |
| POST | `/api/v1/recurrence-rules/{id}/generate` | Сгенерировать задачу для правила на дату |
| POST | `/api/v1/recurrence-rules/generate` | Сгенерировать задачи для всех активных правил |

## Реализованная функциональность

### Типы периодичности

| Тип | Описание | Обязательное поле |
|-----|----------|-------------------|
| `daily` | Каждый N-й день начиная с `start_date` | `every_n_days` |
| `monthly` | Определённое число каждого месяца | `day_of_month` (1-31) |
| `specific_dates` | Конкретные даты | `specific_dates` (массив дат) |
| `even_odd` | Чётные или нечётные числа месяца | `even_odd` ("even" / "odd") |

### Примеры

**Ежедневный обзвон пациентов:**

```bash
curl -X POST http://localhost:8080/api/v1/recurrence-rules \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Обзвон пациентов",
    "description": "Ежедневный обзвон пациентов палаты №3",
    "recurrence_type": "daily",
    "every_n_days": 1,
    "start_date": "2026-04-01"
  }'
```

**Формирование отчётности по чётным дням:**

```bash
curl -X POST http://localhost:8080/api/v1/recurrence-rules \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Формирование отчётности",
    "description": "Отчётность по расходным материалам",
    "recurrence_type": "even_odd",
    "even_odd": "even",
    "start_date": "2026-04-01"
  }'
```

**Еженедельная инвентаризация (конкретные даты):**

```bash
curl -X POST http://localhost:8080/api/v1/recurrence-rules \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Инвентаризация",
    "description": "Инвентаризация склада медикаментов",
    "recurrence_type": "specific_dates",
    "specific_dates": ["2026-04-10", "2026-04-17", "2026-04-24"],
    "start_date": "2026-04-01"
  }'
```

**Ежемесячная задача на 15-е число:**

```bash
curl -X POST http://localhost:8080/api/v1/recurrence-rules \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Ежемесячный отчёт",
    "recurrence_type": "monthly",
    "day_of_month": 15,
    "start_date": "2026-04-01"
  }'
```

**Генерация задач на дату:**

```bash
curl -X POST http://localhost:8080/api/v1/recurrence-rules/generate \
  -H "Content-Type: application/json" \
  -d '{"date": "2026-04-08"}'
```

## Архитектура решения

Проект следует принципам Clean Architecture:

```
internal/
├── domain/
│   ├── task/          # Доменная модель задачи
│   └── recurrence/    # Доменная модель правила периодичности
├── usecase/
│   ├── task/          # Бизнес-логика задач
│   └── recurrence/    # Бизнес-логика периодичности + генерация
├── repository/
│   └── postgres/      # Работа с PostgreSQL
├── transport/
│   └── http/          # HTTP-обработчики и роутинг
├── infrastructure/
│   └── postgres/      # Подключение к БД
└── scheduler/         # Фоновый планировщик
```

### Генерация задач

1. **Автоматически** — фоновый планировщик запускается при старте приложения и с настраиваемым интервалом (по умолчанию 1 час) генерирует задачи по всем активным правилам на текущую дату.

2. **Вручную** — через API `/generate` можно указать конкретную дату для генерации задач. Удобно для тестирования и для создания задач заранее.

### Защита от дублирования

Уникальный ключ `(recurrence_rule_id, scheduled_date)` в таблице `generated_tasks` гарантирует, что для каждого правила на каждую дату создаётся не более одной задачи. Повторный вызов генерации безопасен.

## Проектные решения и предположения

1. **Правило периодичности — шаблон задачи.** Каждое правило содержит `title` и `description`, которые копируются в создаваемую задачу. Задача всегда создаётся со статусом `new`.

2. **Интервал `every_n_days` отсчитывается от `start_date`.** Если `start_date = 2026-04-01` и `every_n_days = 3`, задачи создаются 1, 4, 7, 10 апреля и т.д.

3. **`day_of_month` допускает значения 1-31.** Если указано 31, а в месяце 30 дней — задача за этот месяц не создастся. Это предсказуемое поведение.

4. **`end_date` — опционально.** Без `end_date` правило действует бессрочно. Деактивировать можно через `is_active = false`.

5. **Удаление правила** каскадно удаляет записи из `generated_tasks`, но сами задачи остаются — они уже могут быть в работе.

6. **Чётные/нечётные дни** определяются по числу месяца: 2, 4, 6... — чётные; 1, 3, 5... — нечётные.

7. **Планировщик генерирует задачи только на текущую дату.** Для генерации на произвольную дату используйте API.

package main

import (
	"context"
	"fmt"
	"log/slog"
	"net/http"
	"os"
	"os/signal"
	"syscall"
	"time"

	infrastructurepostgres "example.com/taskservice/internal/infrastructure/postgres"
	postgresrepo "example.com/taskservice/internal/repository/postgres"
	"example.com/taskservice/internal/scheduler"
	transporthttp "example.com/taskservice/internal/transport/http"
	swaggerdocs "example.com/taskservice/internal/transport/http/docs"
	httphandlers "example.com/taskservice/internal/transport/http/handlers"
	recurrenceusecase "example.com/taskservice/internal/usecase/recurrence"
	"example.com/taskservice/internal/usecase/task"
)

func main() {
	logger := slog.New(slog.NewTextHandler(os.Stdout, &slog.HandlerOptions{
		Level: slog.LevelInfo,
	}))

	cfg := loadConfig()

	ctx, stop := signal.NotifyContext(context.Background(), syscall.SIGINT, syscall.SIGTERM)
	defer stop()

	pool, err := infrastructurepostgres.Open(ctx, cfg.DatabaseDSN)
	if err != nil {
		logger.Error("open postgres", "error", err)
		os.Exit(1)
	}
	defer pool.Close()

	taskRepo := postgresrepo.New(pool)
	recurrenceRepo := postgresrepo.NewRecurrenceRepository(pool)

	taskUsecase := task.NewService(taskRepo)
	recurrenceUsecase := recurrenceusecase.NewService(recurrenceRepo, taskRepo, logger)

	taskHandler := httphandlers.NewTaskHandler(taskUsecase)
	recurrenceHandler := httphandlers.NewRecurrenceHandler(recurrenceUsecase)
	docsHandler := swaggerdocs.NewHandler()
	router := transporthttp.NewRouter(taskHandler, recurrenceHandler, docsHandler)

	taskScheduler := scheduler.New(recurrenceUsecase, cfg.SchedulerInterval, logger)
	go taskScheduler.Run(ctx)

	server := &http.Server{
		Addr:              cfg.HTTPAddr,
		Handler:           router,
		ReadHeaderTimeout: 5 * time.Second,
	}

	go func() {
		<-ctx.Done()

		shutdownCtx, cancel := context.WithTimeout(context.Background(), 10*time.Second)
		defer cancel()

		if err := server.Shutdown(shutdownCtx); err != nil {
			logger.Error("shutdown http server", "error", err)
		}
	}()

	logger.Info("http server started", "addr", cfg.HTTPAddr)

	if err := server.ListenAndServe(); err != nil && err != http.ErrServerClosed {
		logger.Error("listen and serve", "error", err)
		os.Exit(1)
	}
}

type config struct {
	HTTPAddr          string
	DatabaseDSN       string
	SchedulerInterval time.Duration
}

func loadConfig() config {
	interval, err := time.ParseDuration(envOrDefault("SCHEDULER_INTERVAL", "1h"))
	if err != nil {
		interval = time.Hour
	}

	cfg := config{
		HTTPAddr:          envOrDefault("HTTP_ADDR", ":8080"),
		DatabaseDSN:       envOrDefault("DATABASE_DSN", "postgres://postgres:postgres@localhost:5432/taskservice?sslmode=disable"),
		SchedulerInterval: interval,
	}

	if cfg.DatabaseDSN == "" {
		panic(fmt.Errorf("DATABASE_DSN is required"))
	}

	return cfg
}

func envOrDefault(key, fallback string) string {
	if value := os.Getenv(key); value != "" {
		return value
	}

	return fallback
}

# -- Helpers --
def add_heading_styled(text, level=1):
    h = doc.add_heading(text, level=level)
    for run in h.runs:
        run.font.name = "Times New Roman"
        run.font.color.rgb = RGBColor(0, 0, 0)
        rp = run._element.get_or_add_rPr()
        rf = rp.find(qn("w:rFonts"))
        if rf is None:
            rf = doc.element.makeelement(qn("w:rFonts"), {})
            rp.insert(0, rf)
        rf.set(qn("w:ascii"), "Times New Roman")
        rf.set(qn("w:hAnsi"), "Times New Roman")
        rf.set(qn("w:cs"), "Times New Roman")
    return h


def add_paragraph(text, bold=False):
    p = doc.add_paragraph()
    run = p.add_run(text)
    run.bold = bold
    run.font.name = "Times New Roman"
    run.font.size = Pt(12)
    return p


def add_bullet(text):
    p = doc.add_paragraph(style="List Bullet")
    p.clear()
    run = p.add_run(text)
    run.font.name = "Times New Roman"
    run.font.size = Pt(12)
    return p


def set_cell_font(cell, text, bold=False, size=Pt(11)):
    cell.text = ""
    p = cell.paragraphs[0]
    run = p.add_run(text)
    run.font.name = "Times New Roman"
    run.font.size = size

def shade_cells(row, color="D9E2F3"):

            qn("w:val"): "clear",
            qn("w:color"): "auto",
            qn("w:fill"): color,
        })

package recurrence

import "time"

type RecurrenceType string

const (
	TypeDaily         RecurrenceType = "daily"
	TypeMonthly       RecurrenceType = "monthly"
	TypeSpecificDates RecurrenceType = "specific_dates"
	TypeEvenOdd       RecurrenceType = "even_odd"
)

func (t RecurrenceType) Valid() bool {
	switch t {
	case TypeDaily, TypeMonthly, TypeSpecificDates, TypeEvenOdd:
		return true
	default:
		return false
	}
}

type EvenOddType string

const (
	Even EvenOddType = "even"
	Odd  EvenOddType = "odd"
)

func (e EvenOddType) Valid() bool {
	switch e {
	case Even, Odd:
		return true
	default:
		return false
	}
}

type RecurrenceRule struct {
	ID             int64          `json:"id"`
	Title          string         `json:"title"`
	Description    string         `json:"description"`
	RecurrenceType RecurrenceType `json:"recurrence_type"`
	EveryNDays     int            `json:"every_n_days"`
	DayOfMonth     int            `json:"day_of_month"`
	SpecificDates  []time.Time    `json:"specific_dates"`
	EvenOdd        EvenOddType    `json:"even_odd"`
	StartDate      time.Time      `json:"start_date"`
	EndDate        *time.Time     `json:"end_date"`
	IsActive       bool           `json:"is_active"`
	CreatedAt      time.Time      `json:"created_at"`
	UpdatedAt      time.Time      `json:"updated_at"`
}

func (r *RecurrenceRule) ShouldGenerateForDate(date time.Time) bool {
	if !r.IsActive {
		return false
	}

	d := truncateToDate(date)
	start := truncateToDate(r.StartDate)

	if d.Before(start) {
		return false
	}

	if r.EndDate != nil {
		end := truncateToDate(*r.EndDate)
		if d.After(end) {
			return false
		}
	}

	switch r.RecurrenceType {
	case TypeDaily:
		return r.matchesDaily(d, start)
	case TypeMonthly:
		return r.matchesMonthly(d)
	case TypeSpecificDates:
		return r.matchesSpecificDates(d)
	case TypeEvenOdd:
		return r.matchesEvenOdd(d)
	default:
		return false
	}
}

func (r *RecurrenceRule) matchesDaily(date, start time.Time) bool {
	if r.EveryNDays <= 0 {
		return false
	}

	daysSinceStart := int(date.Sub(start).Hours() / 24)
	return daysSinceStart%r.EveryNDays == 0
}

func (r *RecurrenceRule) matchesMonthly(date time.Time) bool {
	return date.Day() == r.DayOfMonth
}

func (r *RecurrenceRule) matchesSpecificDates(date time.Time) bool {
	for _, sd := range r.SpecificDates {
		if truncateToDate(sd).Equal(date) {
			return true
		}
	}

	return false
}

func (r *RecurrenceRule) matchesEvenOdd(date time.Time) bool {
	day := date.Day()

	switch r.EvenOdd {
	case Even:
		return day%2 == 0
	case Odd:
		return day%2 != 0
	default:
		return false
	}
}

func truncateToDate(t time.Time) time.Time {
	return time.Date(t.Year(), t.Month(), t.Day(), 0, 0, 0, 0, time.UTC)
}

func Open(ctx context.Context, dsn string) (*pgxpool.Pool, error) {
	if dsn == "" {
		return nil, fmt.Errorf("database dsn is empty")
	}

	cfg, err := pgxpool.ParseConfig(dsn)
	if err != nil {
		return nil, err
	}

	pool, err := pgxpool.NewWithConfig(ctx, cfg)
	if err != nil {
		return nil, err
	}

	if err := pool.Ping(ctx); err != nil {
		pool.Close()
		return nil, err
	}

	return pool, nil
}

func marshalDates(dates []time.Time) (string, error) {
	if len(dates) == 0 {
		return "[]", nil
	}

	strs := make([]string, len(dates))
	for i, d := range dates {
		strs[i] = d.Format("2006-01-02")
	}

	data, err := json.Marshal(strs)
	if err != nil {
		return "", err
	}

	return string(data), nil
}

func unmarshalDates(raw string) ([]time.Time, error) {
	if raw == "" || raw == "[]" {
		return nil, nil
	}

	var strs []string
	if err := json.Unmarshal([]byte(raw), &strs); err != nil {
		return nil, err
	}

	dates := make([]time.Time, 0, len(strs))
	for _, s := range strs {
		t, err := time.Parse("2006-01-02", s)
		if err != nil {
			return nil, err
		}

		dates = append(dates, t)
	}

	return dates, nil
}

package scheduler

import (
	"context"
	"log/slog"
	"time"

	taskdomain "example.com/taskservice/internal/domain/task"
)

type TaskGenerator interface {
	GenerateTasksForDate(ctx context.Context, date time.Time) ([]taskdomain.Task, error)
}

type Scheduler struct {
	generator TaskGenerator
	interval  time.Duration
	logger    *slog.Logger
}

func New(generator TaskGenerator, interval time.Duration, logger *slog.Logger) *Scheduler {
	return &Scheduler{
		generator: generator,
		interval:  interval,
		logger:    logger,
	}
}

func (s *Scheduler) Run(ctx context.Context) {
	s.logger.Info("scheduler started", "interval", s.interval.String())

	s.generate(ctx)

	ticker := time.NewTicker(s.interval)
	defer ticker.Stop()

	for {
		select {
		case <-ctx.Done():
			s.logger.Info("scheduler stopped")
			return
		case <-ticker.C:
			s.generate(ctx)
		}
	}
}

func (s *Scheduler) generate(ctx context.Context) {
	today := time.Now().UTC().Truncate(24 * time.Hour)

	tasks, err := s.generator.GenerateTasksForDate(ctx, today)
	if err != nil {
		s.logger.Error("failed to generate tasks", "error", err, "date", today.Format("2006-01-02"))
		return
	}

	if len(tasks) > 0 {
		s.logger.Info("tasks generated", "count", len(tasks), "date", today.Format("2006-01-02"))
	}
}

	dto := recurrenceRuleDTO{
		ID:             rule.ID,
		Title:          rule.Title,
		Description:    rule.Description,
		RecurrenceType: string(rule.RecurrenceType),
		EveryNDays:     rule.EveryNDays,
		DayOfMonth:     rule.DayOfMonth,
		EvenOdd:        string(rule.EvenOdd),
		StartDate:      rule.StartDate.Format("2006-01-02"),
		IsActive:       rule.IsActive,
		CreatedAt:      rule.CreatedAt.Format(time.RFC3339),
		UpdatedAt:      rule.UpdatedAt.Format(time.RFC3339),
	}

	if rule.EndDate != nil {
		dto.EndDate = rule.EndDate.Format("2006-01-02")
	}

	if len(rule.SpecificDates) > 0 {
		dto.SpecificDates = make([]string, len(rule.SpecificDates))
		for i, d := range rule.SpecificDates {
			dto.SpecificDates[i] = d.Format("2006-01-02")
		}
	}

	return dto
}

func parseDateString(s string) (time.Time, error) {
	return time.Parse("2006-01-02", s)
}

func parseDateStrings(strs []string) ([]time.Time, error) {
	if len(strs) == 0 {
		return nil, nil
	}

	dates := make([]time.Time, 0, len(strs))
	for _, s := range strs {
		d, err := parseDateString(s)
		if err != nil {
			return nil, err
		}

		dates = append(dates, d)
	}

	return dates, nil
}

func (s *Service) generateForRule(ctx context.Context, rule *recurrencedomain.RecurrenceRule, date time.Time) (*taskdomain.Task, error) {
	if !rule.ShouldGenerateForDate(date) {
		return nil, nil
	}

	alreadyGenerated, err := s.repo.IsTaskGenerated(ctx, rule.ID, date)
	if err != nil {
		return nil, err
	}

	if alreadyGenerated {
		return nil, nil
	}

	now := s.now()
	task := &taskdomain.Task{
		Title:       rule.Title,
		Description: rule.Description,
		Status:      taskdomain.StatusNew,
		CreatedAt:   now,
		UpdatedAt:   now,
	}

	created, err := s.taskRepo.Create(ctx, task)
	if err != nil {
		return nil, err
	}

	if err := s.repo.SaveGeneratedTask(ctx, rule.ID, created.ID, date); err != nil {
		return nil, err
	}

	return created, nil
}

func validateCreateInput(input CreateInput) (CreateInput, error) {
	input.Title = strings.TrimSpace(input.Title)
	input.Description = strings.TrimSpace(input.Description)

	if input.Title == "" {
		return CreateInput{}, fmt.Errorf("%w: title is required", ErrInvalidInput)
	}

	if !input.RecurrenceType.Valid() {
		return CreateInput{}, fmt.Errorf("%w: invalid recurrence type", ErrInvalidInput)
	}

	if input.StartDate.IsZero() {
		return CreateInput{}, fmt.Errorf("%w: start date is required", ErrInvalidInput)
	}

	if input.EndDate != nil && input.EndDate.Before(input.StartDate) {
		return CreateInput{}, fmt.Errorf("%w: end date must be after start date", ErrInvalidInput)
	}

	if err := validateTypeSpecificFields(input.RecurrenceType, input.EveryNDays, input.DayOfMonth, input.SpecificDates, input.EvenOdd); err != nil {
		return CreateInput{}, err
	}

	return input, nil
}

func validateUpdateInput(input UpdateInput) (UpdateInput, error) {
	input.Title = strings.TrimSpace(input.Title)
	input.Description = strings.TrimSpace(input.Description)

	if input.Title == "" {
		return UpdateInput{}, fmt.Errorf("%w: title is required", ErrInvalidInput)
	}

	if !input.RecurrenceType.Valid() {
		return UpdateInput{}, fmt.Errorf("%w: invalid recurrence type", ErrInvalidInput)
	}

	if input.StartDate.IsZero() {
		return UpdateInput{}, fmt.Errorf("%w: start date is required", ErrInvalidInput)
	}

	if input.EndDate != nil && input.EndDate.Before(input.StartDate) {
		return UpdateInput{}, fmt.Errorf("%w: end date must be after start date", ErrInvalidInput)
	}

	if err := validateTypeSpecificFields(input.RecurrenceType, input.EveryNDays, input.DayOfMonth, input.SpecificDates, input.EvenOdd); err != nil {
		return UpdateInput{}, err
	}

	return input, nil
}

func validateTypeSpecificFields(rt recurrencedomain.RecurrenceType, everyN int, dayOfMonth int, specificDates []time.Time, evenOdd recurrencedomain.EvenOddType) error {
	switch rt {
	case recurrencedomain.TypeDaily:
		if everyN <= 0 {
			return fmt.Errorf("%w: every_n_days must be positive for daily recurrence", ErrInvalidInput)
		}
	case recurrencedomain.TypeMonthly:
		if dayOfMonth < 1 || dayOfMonth > 31 {
			return fmt.Errorf("%w: day_of_month must be between 1 and 31", ErrInvalidInput)
		}
	case recurrencedomain.TypeSpecificDates:
		if len(specificDates) == 0 {
			return fmt.Errorf("%w: at least one date is required for specific_dates recurrence", ErrInvalidInput)
		}
	case recurrencedomain.TypeEvenOdd:
		if !evenOdd.Valid() {
			return fmt.Errorf("%w: even_odd must be 'even' or 'odd'", ErrInvalidInput)
		}
	}

	return nil
}

package task

import (
	"context"
	"fmt"
	"strings"
	"time"

	taskdomain "example.com/taskservice/internal/domain/task"
)

type Service struct {
	repo Repository
	now  func() time.Time
}

func NewService(repo Repository) *Service {
	return &Service{
		repo: repo,
		now:  func() time.Time { return time.Now().UTC() },
	}
}

func (s *Service) Create(ctx context.Context, input CreateInput) (*taskdomain.Task, error) {
	normalized, err := validateCreateInput(input)
	if err != nil {
		return nil, err
	}

	model := &taskdomain.Task{
		Title:       normalized.Title,
		Description: normalized.Description,
		Status:      normalized.Status,
	}
	now := s.now()
	model.CreatedAt = now
	model.UpdatedAt = now

	created, err := s.repo.Create(ctx, model)
	if err != nil {
		return nil, err
	}

	return created, nil
}

func (s *Service) GetByID(ctx context.Context, id int64) (*taskdomain.Task, error) {
	if id <= 0 {
		return nil, fmt.Errorf("%w: id must be positive", ErrInvalidInput)
	}

	return s.repo.GetByID(ctx, id)
}

func (s *Service) Update(ctx context.Context, id int64, input UpdateInput) (*taskdomain.Task, error) {
	if id <= 0 {
		return nil, fmt.Errorf("%w: id must be positive", ErrInvalidInput)
	}

	normalized, err := validateUpdateInput(input)
	if err != nil {
		return nil, err
	}

	model := &taskdomain.Task{
		ID:          id,
		Title:       normalized.Title,
		Description: normalized.Description,
		Status:      normalized.Status,
		UpdatedAt:   s.now(),
	}

	updated, err := s.repo.Update(ctx, model)
	if err != nil {
		return nil, err
	}

	return updated, nil
}

func (s *Service) Delete(ctx context.Context, id int64) error {
	if id <= 0 {
		return fmt.Errorf("%w: id must be positive", ErrInvalidInput)
	}

	return s.repo.Delete(ctx, id)
}

func (s *Service) List(ctx context.Context) ([]taskdomain.Task, error) {
	return s.repo.List(ctx)
}

func validateCreateInput(input CreateInput) (CreateInput, error) {
	input.Title = strings.TrimSpace(input.Title)
	input.Description = strings.TrimSpace(input.Description)

	if input.Title == "" {
		return CreateInput{}, fmt.Errorf("%w: title is required", ErrInvalidInput)
	}

	if input.Status == "" {
		input.Status = taskdomain.StatusNew
	}

	if !input.Status.Valid() {
		return CreateInput{}, fmt.Errorf("%w: invalid status", ErrInvalidInput)
	}

	return input, nil
}

func validateUpdateInput(input UpdateInput) (UpdateInput, error) {
	input.Title = strings.TrimSpace(input.Title)
	input.Description = strings.TrimSpace(input.Description)

	if input.Title == "" {
		return UpdateInput{}, fmt.Errorf("%w: title is required", ErrInvalidInput)
	}

	if !input.Status.Valid() {
		return UpdateInput{}, fmt.Errorf("%w: invalid status", ErrInvalidInput)
	}

	return input, nil
}

    day_of_month INT CHECK (day_of_month IS NULL OR (day_of_month >= 1 AND day_of_month <= 31)),
    specific_dates TEXT DEFAULT '[]',
    even_odd TEXT CHECK (even_odd IS NULL OR even_odd IN ('even', 'odd')),
    start_date DATE NOT NULL,
    end_date DATE,
    is_active BOOLEAN NOT NULL DEFAULT TRUE,
    created_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    updated_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

CREATE INDEX IF NOT EXISTS idx_recurrence_rules_active ON recurrence_rules (is_active) WHERE is_active = TRUE;

CREATE TABLE IF NOT EXISTS generated_tasks (
    recurrence_rule_id BIGINT NOT NULL REFERENCES recurrence_rules(id) ON DELETE CASCADE,
    task_id BIGINT NOT NULL REFERENCES tasks(id) ON DELETE CASCADE,
    scheduled_date DATE NOT NULL,
    created_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    PRIMARY KEY (recurrence_rule_id, scheduled_date)
);

CREATE INDEX IF NOT EXISTS idx_generated_tasks_task_id ON generated_tasks (task_id);
